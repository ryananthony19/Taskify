# Taskify

A React-based task management application that enables users to add, edit, delete, and track tasks efficiently with a clean UI and real-time updates.

## 🚀 Setup & Run

1. Clone the repository
git clone https://github.com/ryananthony19/Taskify.git

2. Navigate into the project
cd Taskify

3. Install dependencies
npm install

4. Start the development server
npm run dev
